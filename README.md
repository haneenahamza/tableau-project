# HR Analytics Dashboard using Tableau

## Project Overview
Developed an interactive HR Analytics dashboard using Tableau to analyze employee trends and workforce metrics.

## Objective
To identify patterns affecting employee attrition and understand workforce demographics for better decision-making.

## Tools Used
- Tableau
- Excel / CSV
- Data Cleaning
- Data Visualization

## KPIs
- Total Employees
- Attrition Rate
- Active Employees
- Average Age
- Department-wise Attrition
- Job Satisfaction Score

## Dashboard Visualizations
- KPI Cards
- Attrition Analysis
- Department Distribution
- Gender Analysis
- Age Group Analysis
- Interactive Filters

## Key Insights
- Identified departments with high attrition
- Analyzed employee demographics
- Found trends related to employee retention
